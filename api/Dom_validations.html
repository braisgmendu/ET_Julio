Class: Dom_validations

Dom_validations()

new Dom_validations()

Clase que maneja las validaciones del DOM, extendiendo la funcionalidad de la clase Dom. Esta clase implementa un sistema completo de validación de formularios que incluye: - Validaciones atómicas (tamaño mínimo/máximo, expresiones regulares) - Validaciones de archivos (tipo, tamaño, formato de nombre) - Validaciones especiales por campo - Validaciones específicas por acción (ADD, EDIT, SEARCH, DELETE)
