Class: Dom

Dom()

new Dom()

Clase principal que maneja la manipulación del DOM y la interfaz de usuario. Esta clase es responsable de crear y gestionar formularios, tablas, modales y otros elementos de la interfaz. Proporciona funcionalidades para la creación dinámica de formularios basados en estructuras de datos, manejo de tablas de datos, gestión de errores y visualización de resultados de tests.
