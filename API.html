Clases para Validaciones

Validaciones_Atomicas

Clase que contiene las validaciones atómicas para los campos del formulario. Proporciona métodos para validar tamaños, expresiones regulares, tipos de archivo y formatos.

constructor()

Constructor de la clase Validaciones_Atomicas

min_size(id, parametro)

Valida que el tamaño del campo sea mayor o igual al parámetro especificado

Parámetro Tipo Descripción
id string ID del campo a validar
parametro number Tamaño mínimo permitido
Retorna: boolean - true si el tamaño es válido, false en caso contrario
max_size(campo, param)

Valida que el tamaño del campo sea menor o igual al parámetro especificado

Parámetro Tipo Descripción
campo string ID del campo a validar
param number Tamaño máximo permitido
Retorna: boolean - true si el tamaño es válido, false en caso contrario
reg_exp(campo, param)

Valida que el campo cumpla con la expresión regular especificada

Parámetro Tipo Descripción
campo string ID del campo a validar
param string Expresión regular a validar
Retorna: boolean - true si el campo cumple con la expresión regular, false en caso contrario
max_size_file(objfile, maxsize)

Valida que el tamaño del archivo sea menor o igual al parámetro especificado

Parámetro Tipo Descripción
objfile File Archivo a validar
maxsize number Tamaño máximo permitido en bytes
Retorna: boolean - true si el tamaño es válido, false en caso contrario
file_type(objfile, array_tipos)

Valida que el tipo de archivo esté en la lista de tipos permitidos

Parámetro Tipo Descripción
objfile File Archivo a validar
array_tipos Array<string> Lista de tipos MIME permitidos
Retorna: boolean - true si el tipo es válido, false en caso contrario

Dom_validations

Extiende de: Dom

Clase que maneja las validaciones del DOM, extendiendo la funcionalidad de la clase Dom. Gestiona la validación de campos del formulario según las reglas definidas para cada acción.

constructor(validaciones, entidad)

Constructor de la clase Dom_validations

Parámetro Tipo Descripción
validaciones Validaciones_Atomicas Instancia de la clase de validaciones atómicas
entidad string Nombre de la entidad a validar
load_validations(action)

Carga las validaciones definidas para una acción específica

Parámetro Tipo Descripción
action string Acción para la que cargar las validaciones (ADD, EDIT, SEARCH, DELETE)
comprobarCampo(campo, action)

Valida un campo según las reglas definidas para la acción especificada

Parámetro Tipo Descripción
campo string ID del campo a validar
action string Acción que se está realizando (ADD, EDIT, SEARCH, DELETE)
Retorna: boolean - true si el campo es válido, false en caso contrario
