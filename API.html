Documentación de la API

Dom

Clase principal que maneja la manipulación del DOM y la interfaz de usuario. Esta clase es responsable de crear y gestionar formularios, tablas, modales y otros elementos de la interfaz.

createForm(action, estructura)

Crea un formulario dinámico basado en la estructura proporcionada.
Parámetros:
  • action (string): Acción a realizar (ADD, EDIT, SEARCH, DELETE)
  • estructura (object): Estructura de la entidad

hacertabla()

Genera una tabla HTML con los datos proporcionados.

Dom_validations

Clase que maneja las validaciones de formularios y campos.

validarFormulario()

Valida todos los campos del formulario actual.
Retorna: boolean - true si todos los campos son válidos

Validaciones_Atomicas

Clase que proporciona validaciones atómicas para los campos de formulario.

min_size(id, parametro)

Valida que el tamaño del campo sea mayor o igual al parámetro especificado.
Parámetros:
  • id (string): ID del campo a validar
  • parametro (number): Tamaño mínimo permitido
Retorna: boolean - true si el tamaño es válido

max_size(campo, param)

Valida que el tamaño del campo sea menor o igual al parámetro especificado.
Parámetros:
  • campo (string): ID del campo a validar
  • param (number): Tamaño máximo permitido
Retorna: boolean - true si el tamaño es válido

reg_exp(campo, param)

Valida que el campo cumpla con la expresión regular especificada.
Parámetros:
  • campo (string): ID del campo a validar
  • param (string): Expresión regular a validar
Retorna: boolean - true si el campo cumple con la expresión regular

test_IU

Clase que maneja la ejecución y validación de tests de la interfaz de usuario.

test_run()

Ejecuta la suite de tests.
Retorna: Promise - Resultado de la ejecución de los tests

verificarPruebas()

Verifica los resultados de las pruebas ejecutadas.
Retorna: boolean - true si todas las pruebas pasan

Entidad_Abstracta

Clase base abstracta que define la estructura y comportamiento común para todas las entidades del sistema.

constructor(entidad, def_estructura)

Inicializa una nueva instancia de la entidad abstracta.
Parámetros:
  • entidad (string): Nombre de la entidad
  • def_estructura (object): Definición de la estructura de la entidad

change_value_IU(atributo, valor)

Método para transformar valores antes de mostrarlos en la interfaz. Puede ser sobrescrito por las clases hijas para implementar transformaciones específicas.
Parámetros:
  • atributo (string): Nombre del atributo
  • valor (any): Valor a transformar
Retorna: any - Valor transformado (por defecto retorna el valor original)

createFormGenerico(action, parametros)

Crea un formulario genérico para una acción específica.
Parámetros:
  • action (string): Acción a realizar (ADD, EDIT, DELETE, SEARCH)
  • parametros (object): Parámetros opcionales para cargar en el formulario

ADD()

Realiza la operación de añadir un nuevo registro.
Retorna: Promise - Promesa que se resuelve cuando la operación se completa

EDIT()

Realiza la operación de editar un registro existente.
Retorna: Promise - Promesa que se resuelve cuando la operación se completa

DELETE()

Realiza la operación de eliminar un registro.
Retorna: Promise - Promesa que se resuelve cuando la operación se completa

SEARCH()

Realiza una búsqueda en la entidad y actualiza la visualización de los resultados.
Retorna: Promise - Promesa que se resuelve cuando la búsqueda se completa

ExternalAccess

Clase que maneja las comunicaciones con el backend y el acceso a recursos externos.

back_request(action, data)

Realiza una petición al backend.
Parámetros:
  • action (string): Acción a realizar
  • data (object): Datos a enviar
Retorna: Promise - Resultado de la petición

analysis_preparation

Clase que maneja la preparación y análisis de datos. Extiende de Entidad_Abstracta.

constructor()

Inicializa una nueva instancia de analysis_preparation.

change_value_IU(atributo, valoratributo)

Sobrescribe el método de la clase padre para transformar valores específicos de analysis_preparation. Convierte el nombre del archivo en un enlace clickeable cuando el atributo es 'file_analysis_preparation'.
Parámetros:
  • atributo (string): Nombre del atributo
  • valoratributo (any): Valor a transformar
Retorna: any - Valor transformado o el valor original si no requiere transformación

characteristic

constructor()

Inicializa una nueva instancia de characteristic.

change_value_IU(atributo, valoratributo)

Sobrescribe el método de la clase padre para transformar valores específicos de characteristic. Convierte el nombre del archivo en un enlace clickeable cuando el atributo es 'file_characteristic'.
Parámetros:
  • atributo (string): Nombre del atributo
  • valoratributo (any): Valor a transformar
Retorna: any - Valor transformado o el valor original si no requiere transformación

project

Clase que maneja los proyectos del sistema. Extiende de Entidad_Abstracta.

constructor()

Inicializa una nueva instancia de project.

change_value_IU(atributo, valoratributo)

Sobrescribe el método de la clase padre para transformar valores específicos de project. Convierte el nombre del archivo en un enlace clickeable y formatea las fechas cuando corresponda.
Parámetros:
  • atributo (string): Nombre del atributo
  • valoratributo (any): Valor a transformar
Retorna: any - Valor transformado o el valor original si no requiere transformación

check_special_START_DATE_PROJECT()

Verifica la fecha de inicio del proyecto.
Retorna: boolean|string - true si la fecha es válida, código de error en caso contrario

check_special_END_DATE_PROJECT()

Verifica la fecha de finalización del proyecto.
Retorna: boolean|string - true si la fecha es válida, código de error en caso contrario
